## 1. 软件概述

随声阅（SoundSync）是一款专为阅读爱好者设计的电子书阅读与语音朗读应用。本应用支持多种电子书格式导入，并提供文本转语音(TTS)朗读和语音合成功能，让用户在不方便阅读时也能享受听书体验。主要特点包括：

- 多格式电子书支持（EPUB、PDF、TXT、MOBI、MD、DOC/DOCX等）
- 网页内容导入与阅读
- 文本创建与编辑
- 实时TTS朗读
- 离线语音合成
- 阅读进度自动保存
- 书架管理与分类浏览
- 自定义阅读体验（字体大小、行间距等）
- 语音参数调节（语速、音调、音量）
- 支持设置语音文件为闹钟铃声

## 2. 安装说明

### 系统要求
- 操作系统：Android 6.0 (API 23)及以上版本
- 存储空间：至少100MB可用空间
- 运行内存：至少2GB RAM

### 安装步骤
1. 从应用商店下载"随声阅"应用
2. 点击安装包进行安装
3. 根据系统提示完成安装过程
4. 安装完成后，点击应用图标启动随声阅

### 权限说明
首次启动时，应用将请求以下必要权限：
- 存储权限：用于访问本地电子书文件和保存合成语音文件
- 摄像头权限：用于拍照导入文本功能
- 闹钟权限：用于设置语音文件作为闹钟铃声

## 3. 功能介绍

### 3.1 主界面

随声阅应用的主界面由以下几个部分组成：

1. **顶部导航栏**：显示当前页面标题和操作按钮
2. **底部导航菜单**：包含四个主要功能区域的入口
   - 书架：显示已导入的电子书，按添加日期排序
   - 书库：浏览和管理应用创建的所有文件
   - 历史：查看最近阅读记录，按最后阅读时间排序
   - 设置：调整应用配置和偏好设置

#### 底部导航菜单使用说明
- 点击对应图标可切换到相应功能区域
- 当前所在区域的图标会高亮显示
- 导航菜单在应用内所有主要页面均可见
- 在阅读页面，导航菜单会自动隐藏，点击屏幕中央可呼出

### 3.2 书架管理

书架是随声阅应用的默认主页面，用于管理和浏览所有导入的电子书。

#### 书架界面组成
- **顶部导航栏**：左侧显示"我的书架"标题，右侧为搜索图标和菜单按钮
- **书籍列表**：以网格或列表形式展示书籍
- **底部导航栏**：用于切换主要功能区域

#### 书架功能
1. **添加书籍**：点击右上角菜单按钮，选择以下导入方式
   - 新建文本：创建自定义文本内容
   - 网址导入：从网页导入文章内容
   - 本地导入：从设备存储中选择电子书文件
   - 拍照导入：通过拍照或相册选择图片进行OCR文字识别

2. **书籍排序**：点击菜单按钮，选择"书籍排序"，可选择按添加日期、书名等方式排序

3. **搜索书籍**：点击搜索图标，输入关键词搜索书架中的书籍

4. **批量操作**：
   - 长按任意书籍进入批量操作模式
   - 或点击右上角菜单中的"批量操作"选项
   - 进入批量模式后，可选择多本书籍进行操作
   - 批量删除：选择书籍后点击删除图标，可选择是否同时删除书籍文件
   - 全选/取消全选：快速选择或取消选择所有书籍
   - 取消：退出批量操作模式

#### 书籍信息显示
每本书籍显示以下信息：
- 封面图片
- 书名
- 作者（如有）
- 添加日期或最后阅读时间（在历史列表中）

### 3.3 电子书导入

随声阅提供多种方式导入电子书内容，详细操作步骤如下：

#### 本地导入
1. 点击书架页面右上角菜单按钮
2. 选择"本地导入"选项
3. 在系统文件选择器中浏览并选择要导入的电子书文件
4. 支持的文件格式：EPUB、PDF、TXT、MOBI、MD、DOC/DOCX等
5. 文件导入后会自动提取元数据并生成封面

**PDF文件特别说明**：
- 导入PDF文件时，系统会自动提取文本内容并保存为TXT文件
- 提取的文本文件与原PDF文件同名且保存在相同目录
- 系统保存原始PDF文件路径，确保文件溯源关系完整
- 用户可以像阅读TXT文件一样阅读提取的文本内容

#### 网址导入
1. 点击书架页面右上角菜单按钮
2. 选择"网址导入"选项
3. 在弹出的对话框中输入要导入的网页URL
4. 点击"导入"按钮，系统会自动获取并处理网页内容
5. 导入完成后，网页内容会以文本形式保存，并添加到书架

**网址导入功能说明**：
- 自动提取网页中的主要文章内容
- 保存的文件以网页标题作为书名
- 保存原始URL，支持稍后重新访问原网页
- 网页内容格式化处理，提供更好的阅读体验

#### 新建文本
1. 点击书架页面右上角菜单按钮
2. 选择"新建文本"选项
3. 在弹出的文本编辑器中输入内容
4. 点击"保存"按钮完成创建
5. 新建的文本将以第一行内容作为标题，并添加到书架中

#### 拍照导入
1. 点击书架页面右上角菜单按钮
2. 选择"拍照导入"选项
3. 选择"拍照"或"从相册选择"
4. 拍照或选择图片后，进入图片裁剪界面
5. 调整裁剪区域以精确选择需要识别的文本部分
6. 点击确认按钮，系统会通过OCR识别图片中的文字
7. 识别完成后，可以预览并编辑识别结果
8. 点击"保存"按钮，识别的文本将作为新书籍添加到书架

### 3.4 阅读功能

随声阅提供统一的阅读界面，支持多种电子书格式，用户体验一致。

#### 阅读界面
阅读界面有两种状态：
1. **全屏阅读界面**
   - 屏幕大部分区域显示书籍内容
   - 底部显示状态栏，包含书名和页码/进度百分比
   - 点击屏幕中央区域可切换到带工具栏的阅读界面

2. **带工具栏的阅读界面**
   - 顶部工具栏：包含返回按钮、章节标题和菜单按钮
   - 底部工具栏：包含翻页控制、进度条和功能按钮
   - 再次点击屏幕任意区域可返回全屏阅读界面

#### 阅读操作
1. **翻页**：
   - 全屏模式：点击屏幕左侧区域上一页，右侧区域下一页
   - 工具栏模式：使用底部工具栏的左右箭头或进度条

2. **阅读设置**：点击底部工具栏的设置图标，可调整
   - 字体大小
   - 行间距
   - TTS语速

3. **目录查看**：点击底部工具栏的目录图标或顶部菜单中的"查看目录"选项

4. **文本编辑**：点击顶部菜单中的"修改文本"选项，可直接编辑当前书籍内容

5. **打开网址**：对于通过网址导入的内容，点击顶部菜单中的"打开网址"选项可在应用内浏览原网页

6. **分享内容**：点击顶部菜单中的"分享内容"选项，可将当前阅读内容分享至其他应用

### 3.5 TTS朗读功能

随声阅的TTS朗读功能可以将书籍内容转换为语音实时朗读，支持后台播放和自动翻页。

#### 使用方法
1. 在阅读界面点击底部工具栏的"朗读"图标或顶部菜单中的"开始朗读"选项
2. 系统将从当前页开始朗读文本内容
3. 朗读过程中当前朗读的句子会高亮显示（当页面同步时）
4. 朗读到页面末尾时自动翻页并继续朗读

#### 朗读控制
当TTS朗读开始后，屏幕上会显示音频播放控件，提供以下功能：
- 暂停/继续：控制朗读的暂停和继续
- 上一句/下一句：跳转到上一句或下一句内容
- 停止：完全停止朗读
- 边听边看：当朗读页面与当前浏览页面不同时，点击可跳转到朗读位置

#### 后台朗读
TTS朗读功能支持后台运行，特点如下：
1. 退出阅读界面或返回书架，朗读仍会继续
2. 朗读状态会在通知栏显示，并提供控制按钮
3. 在书架界面底部会显示朗读控制栏
4. 点击通知或控制栏的"边听边看"可直接返回到朗读位置

### 3.6 语音合成功能

语音合成功能可将书籍内容合成为MP3音频文件，便于离线收听。

#### 使用方法
1. 在阅读界面点击底部工具栏的"合成语音"图标或顶部菜单中的"合成语音"选项
2. 在弹出的合成设置窗口中选择：
   - 合成范围：当前页或当前章节
   - 语速设置：调整语音播放速度
   - 音调设置：调整语音音调
   - 音量设置：调整语音音量
3. 点击"开始合成"按钮启动合成过程
4. 合成过程中会显示进度百分比
5. 合成完成后，会显示成功提示和文件保存位置

#### 语音文件存储
- 合成的语音文件保存在设备的Documents/SoundSync/voices目录下
- 文件命名格式：书名前18个字符_年月日+随机数.mp3
- 文件可在应用内的语音列表中管理和播放
- 即使卸载应用，语音文件也会保留在设备存储中

### 3.7 语音文件管理

随声阅提供专门的语音文件管理功能，用于播放、重命名和删除合成的语音文件。

#### 语音列表
1. 在阅读界面点击底部工具栏的"语音列表"图标或顶部菜单中的"语音列表"选项
2. 或在底部导航菜单中选择"书库"，然后点击"语音文件"分类
3. 语音文件按照生成时间倒序排列，每条记录显示：
   - 文件名称
   - 音频时长
   - 生成时间
   - 文件大小

#### 语音播放
点击语音文件后，底部会显示播放控制栏，提供以下功能：
- 播放/暂停：控制音频播放
- 快退/快进：向前或向后跳转15秒
- 进度条：显示播放进度，可拖动调整位置
- 时间显示：显示当前播放时间和总时长

#### 文件操作
语音文件支持以下操作：
1. **重命名**：点击播放控制栏中的重命名图标
2. **删除**：点击播放控制栏中的删除图标，删除前会显示确认提示
3. **查看所在目录**：点击三点菜单中的"所属目录"选项，打开系统文件管理器并定位到文件
4. **设置为闹钟铃声**：点击三点菜单中的"设置闹钟"选项，将语音文件设置为系统闹钟铃声

### 3.8 设置中心

设置中心提供应用的各种配置选项，通过底部导航菜单的"设置"图标进入。

#### 主要设置项
1. **语言设置**：切换应用界面语言
2. **主题样式**：设置应用的显示主题（明亮/暗黑）
3. **阅读设置**：
   - 默认字体大小
   - 默认行间距
4. **TTS设置**：
   - 语速调节
   - 测试朗读功能
5. **后台运行设置**：
   - 关闭电池优化：请求系统忽略电池优化，保证TTS服务在后台稳定运行
   - 允许后台运行：针对不同厂商设备优化后台运行权限设置
6. **关于应用**：
   - 应用版本信息
   - 开发者信息
   - 更新日志

## 4. 常见问题

### 电子书导入问题
**问题**：导入电子书后无法打开或内容显示异常  
**解决方案**：
- 确认电子书格式受支持（EPUB、PDF、TXT、MOBI、MD、DOC/DOCX）
- 检查电子书文件是否完整，尝试重新导入
- 对于加密的PDF或受保护的电子书，可能无法正常导入

### TTS朗读问题
**问题**：TTS朗读时声音异常或无法朗读  
**解决方案**：
- 确保设备已安装和启用TTS引擎
- 在系统设置中检查默认TTS引擎设置
- 调整应用内的TTS语速设置
- 重启应用或设备后再试

### 后台运行问题
**问题**：退出应用后TTS朗读自动停止  
**解决方案**：
- 在应用设置中开启"关闭电池优化"选项
- 开启"允许后台运行"选项，并按照指引完成厂商特定设置
- 检查设备的电池管理或应用管理设置，确保随声阅不受限制

### 存储权限问题
**问题**：无法保存或访问电子书/语音文件  
**解决方案**：
- 确保已授予应用必要的存储权限
- 检查设备存储空间是否充足
- 对于Android 10+设备，可能需要使用系统文件管理器授予特定目录访问权限
